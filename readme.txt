This plugin helps you embed angellist startup and investor/founder widgets at the end of your articles.

Here are some screenshots of it in action: http://imgur.com/a/8AxR7

It's still in very early stages of development, so we haven't integrated into wordpress's distribution system yet. You'll have to manually install using the instructions here: http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation

Please email hackers@angel.co with any questions you may have.
